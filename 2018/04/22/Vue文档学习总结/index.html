Vue文档学习总结

  1. 只有当实例被创建时 data 中存在的属性才是响应式的。如果之后需要一个属性，但是一开始它为空或不存在，那么可以设置初始值

  2. 使用 Object.freeze()，会阻止修改现有的属性，也意味着响应系统无法再_追踪_变化

  3. 对于所有的数据绑定，Vue.js 都提供了完全的 JavaScript 表达式支持，每个绑定都只能包含单个表达式，所以语句和流控制不能生效

  6. 我们可以将同一函数定义为一个方法而不是一个计算属性。两种方式的最终结果完全相同。然而，不同的是计算属性基于它们的依赖进行缓存，多次访问计算属性会立即返回之前的计算结果，而不必再次执行函数

  7. v-bind 绑定 class 绑定的数据对象不必内联定义在模板里

  8. 路由页 webpack 懒加载页面语法
