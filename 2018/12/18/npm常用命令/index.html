npm常用命令

1. 初始化 npm init

生成 package.json

npm init -y 或 npm init -yes: 按默认配置生成，不需要选择

2. 安装 / 卸载 npm install/uninstall

接受同样的选项和参数

简写 npm i / npm un

选项：

  • –global: 简写 -g 全局安装
  • –save: 简写 -S 更新包的名称和版本在 package.json 的 dependencies 字段
  • –save-dev: 简写 -D 同上，在devDependencies

3. 更新 npm update

npm info package_name: 查看 package_name 包的信息，可以用来查看是否更新

npm outdated: 检测当前安装的所有npm包是否有更新

npm update package_name: 更新 package_name 包

4. 发布 npm publish

npm login: 登录 npm 账号

npm包的版本号

x.y.z的格式：

  • x 主版本号，通常有重大改变或者达到里程碑才改变

  • y 次要版本号或二级版本号，在保证主体功能基本不变的情况下，如果适当增加了新功能可以更新此版本号

  • z 表示尾版本号或者补丁号，一些小范围的修修补补就可以更新补丁号

更新版本号

  • 手动：修改package.json的version字段
  • 自动：使用 npm version 命令
